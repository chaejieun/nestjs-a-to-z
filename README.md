### 환경 설정
1. node 설치
  `$ npm -v`
   `$ npm i -g npm`
2. yarn 설치
  `$ npm install --global yarn`
3. nodemon, prettier 설치
  `$ yarn global add nodemon`
  `$ yarn global add prettier`
  
  
  ### nodemon
$ nodemon main.js 명령어로 실시간으로 스크립트 파일을 디버깅할 수 있다.
